## Hoofdstuk

Het begrijpend lezen van wetenschappelijke artikelen is bewezen leesmateriaal in het middelbaar onderwijs. Vooral met taalstoornissen kan het moeizaam verlopen. Personaliseerbare tekstvereenvoudiging kan een oplossing bieden. Hoewel er geen eenduidige oplossing is, kan prototype Pentimentor helpen.



## Hoofdstuk

Plaats jezelf in schoenen van scholier 3e graad. Voor opdracht begrijpend lezen, moet je deel uit wetenschappelijk artikel afgebeeld op simulatie. Moeilijker dan gedacht? Symptomen van dyslexie gaan verder dan dansende letters, maar simulatie benadrukt stoorzenders waarmee scholieren kampen. Artikelen geschreven voor specifiek vakdomein.
